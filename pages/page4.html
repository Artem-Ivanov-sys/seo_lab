Умовні оператори в Python

Умовні оператори в Python дозволяють програмі приймати рішення на основі певних умов. Вони дозволяють виконувати різні частини коду залежно від того, чи є умова істинною чи хибною.

1. Основний синтаксис умовних операторів

Python використовує такі умовні оператори для прийняття рішень:

  1. if — основний оператор для перевірки умови.
  2. elif — перевіряє додаткові умови, якщо попередня умова була хибною.
  3. else — виконує блок коду, якщо всі попередні умови були хибними.


if умова:
    # код, який виконується, якщо умова істинна
elif інша_умова:
    # код, який виконується, якщо інша умова істинна
else:
    # код, який виконується, якщо жодна умова не істинна

Приклад базового використання if, elif, else:


age = 20

if age < 18:
    print("Ви неповнолітній.")
elif age == 18:
    print("Вам точно 18 років!")
else:
    print("Ви дорослий.")

Результат:


Ви дорослий.

  1. Якщо змінна age менша за 18, програма виконає блок коду після if.
  2. Якщо змінна дорівнює 18, програма виконає блок після elif.
  3. Якщо жодна з умов не виконана, програма перейде до блоку після else.

2. Оператори порівняння та логічні оператори

Python підтримує широкий набір операторів для перевірки умов.

Оператори порівняння:

  1. == — дорівнює
  2. != — не дорівнює
  3. > — більше
  4. < — менше
  5. >= — більше або дорівнює
  6. <= — менше або дорівнює


x = 10
y = 5

if x > y:
    print("x більше ніж y")
else:
    print("x не більше ніж y")

Результат:


x більше ніж y

Логічні оператори:

Іноді вам потрібно перевірити кілька умов одночасно. Для цього використовуються логічні оператори:

  1. and — всі умови повинні бути істинними.
  2. or — принаймні одна умова повинна бути істинною.
  3. not — інвертує істинність умови.

Приклад з and:


x = 15

if x > 10 and x < 20:
    print("x знаходиться між 10 і 20")

Результат:


x знаходиться між 10 і 20

Приклад з or:


age = 17
has_permission = True

if age >= 18 or has_permission:
    print("Вхід дозволений.")
else:
    print("Вхід заборонено.")

Результат:


Вхід дозволений.
