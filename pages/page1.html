Змінні в Python

Змінні — це фундаментальні компоненти будь-якої мови програмування, зокрема і Python. Змінна — це контейнер для зберігання даних, які можуть змінюватися під час виконання програми. У Python, оголошення змінної не вимагає вказівки типу даних, оскільки Python автоматично визначає тип змінної під час її ініціалізації.

1. Оголошення змінних

У Python змінна створюється під час першого присвоєння їй значення: 


x = 10      # Змінна x отримала значення 10
name = "Alex"  # Змінна name отримала рядкове значення

У цьому прикладі змінна x є цілим числом, а змінна name — рядком. Python автоматично визначає тип змінної на основі присвоєного значення.

2. Правила іменування

Python має кілька простих правил, яких потрібно дотримуватися під час іменування змінних:

  • Ім'я змінної повинно починатися з літери (a-z, A-Z) або символу підкреслення (_).
  • Ім'я не може починатися з цифри.
  • Ім'я змінної може містити лише букви, цифри і символ підкреслення.
  • Python є чутливим до регістру, тому змінні Name і name — це різні змінні.


my_var = 5      # правильне ім'я
_my_var = 10    # також правильне ім'я
myVar2 = "test" # допустимо

2my_var = 10    # неправильне ім'я, починається з цифри
my-var = 10     # неправильне ім'я, містить дефіс

3. Виведення змінних

У Python змінні можна вивести за допомогою оператора print().


my_var = 5      # створюємо змінну
print(my_var)   # виведе 5

4. Множинне присвоєння

У Python можна присвоїти кілька значень кільком змінним одночасно.


a, b, c = 1, 2, 3
