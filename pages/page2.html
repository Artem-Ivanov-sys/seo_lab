Типи даних в Python

Python підтримує різноманітні типи даних, які дозволяють працювати з числами, рядками, списками, словниками та іншими структурами. Пропонуємо розглянемо основні типи даних, їх властивості та приклади використання.

1. Числові типи даних

Python має три основні типи для роботи з числами:

  • int: Цілі числа (наприклад, 1, -5, 100).
  • float: Числа з плаваючою точкою (наприклад, 3.14, -0.001).
  • complex: Комплексні числа (наприклад, 1+2j).


# Ціле число
x = 10
print(type(x))  # <class 'int'>

# Число з плаваючою точкою
y = 3.14
print(type(y))  # <class 'float'>

# Комплексне число
z = 1 + 2j
print(type(z))  # <class 'complex'>

2. Рядки

Рядки в Python — це послідовності символів. Їх можна створювати за допомогою одинарних або подвійних лапок. 


# Рядок із одинарними лапками
s1 = 'Привіт, світ!'
print(s1)

# Рядок із подвійними лапками
s2 = "Python — чудова мова"
print(s2)

# Багаторядковий рядок
s3 = '''Це
багаторядковий
рядок.'''
print(s3)

3. Логічний тип

Тип bool представляє логічні значення: True або False. Логічні операції використовуються для умовних конструкцій та циклів.


# Логічне значення True
is_active = True
print(is_active)  # True

# Логічні операції
print(5 > 3)  # True
print(10 == 10)  # True
print(7 < 3)  # False

4. Списки

Списки — це впорядковані змінні колекції елементів, які можуть містити об'єкти будь-яких типів. Вони визначаються за допомогою квадратних дужок.


# Створення списку
my_list = [1, 2, 3, 'apple', 3.14]
print(my_list)  # [1, 2, 3, 'apple', 3.14]

# Доступ до елементів списку
print(my_list[0])  # 1

# Зміна елемента
my_list[3] = 'banana'
print(my_list)  # [1, 2, 3, 'banana', 3.14]

# Додавання елемента
my_list.append('new item')
print(my_list)  # [1, 2, 3, 'banana', 3.14, 'new item']

5. Словники

Словники — це невпорядковані колекції пар ключ-значення. Вони дозволяють зберігати дані, асоційовані з певними ключами. Словники визначаються за допомогою фігурних дужок.


# Створення словника
my_dict = {'name': 'John', 'age': 25, 'city': 'Kyiv'}
print(my_dict)  # {'name': 'John', 'age': 25, 'city': 'Kyiv'}

# Доступ до значення за ключем
print(my_dict['name'])  # John

# Додавання нового ключа
my_dict['job'] = 'developer'
print(my_dict)  # {'name': 'John', 'age': 25, 'city': 'Kyiv', 'job': 'developer'}

6. None

None — це спеціальний тип даних, який означає відсутність значення. Він часто використовується як початкове значення для змінних або як повертаєме значення функцій, які не мають результату.


# Змінна з None
x = None
print(x)  # None

# Функція, яка нічого не повертає, повертає None
def my_func():
    pass

print(my_func())  # None
