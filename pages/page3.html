Арифметичні операції в Python

Python підтримує широкий набір арифметичних операцій для роботи з числами. Ці операції дозволяють виконувати базові математичні дії, такі як додавання, віднімання, множення, ділення та інші.

1. Основні арифметичні операції

  1. Додавання + Додає два числа. 

    
a = 10
b = 5
result = a + b  # 15
print(result)

  2. Віднімання - Віднімає одне число від іншого. 

    
a = 10
b = 5
result = a - b  # 5
print(result)

  3. Множення * Множить два числа. 

    
a = 10
b = 5
result = a * b  # 50
print(result)

  4. Ділення / Ділить одне число на інше. Повертає результат з плаваючою точкою. 

    
a = 10
b = 3
result = a / b  # 3.3333333333333335
print(result)

  5. Цілочисельне ділення // Ділить два числа і повертає цілу частину від ділення. 

    
a = 10
b = 3
result = a // b  # 3
print(result)

  6. Залишок від ділення (Модуль) % Повертає залишок від ділення двох чисел. 

    
a = 10
b = 3
result = a % b  # 1
print(result)

  7. Піднесення до степеня ** Підносить одне число до степеня іншого. 

    
a = 2
b = 3
result = a ** b  # 8 (2 в степені 3)
print(result)

2. Оператори присвоєння з арифметичними операціями

Python підтримує комбіновані оператори, які дозволяють виконувати операції і присвоєння в одному виразі: 


x = 10
x += 5  # Тепер x = 15
x -= 3  # Тепер x = 12
x *= 2  # Тепер x = 24
x /= 4  # Тепер x = 6.0
print(x)

3. Пріоритет операцій

Python виконує арифметичні операції в порядку, визначеному математичними правилами пріоритету. Пріоритет операцій можна змінити за допомогою круглих дужок.

Порядок виконання операцій:

  1. Дужки ()
  2. Піднесення до степеня **
  3. Множення, ділення, цілочисельне ділення, залишок від ділення *, /, //, %
  4. Додавання і віднімання +, -


result = 10 + 5 * 2  # Спочатку множення, потім додавання: 10 + (5 * 2) = 20
print(result)  # 20

# Використання дужок для зміни порядку операцій
result = (10 + 5) * 2  # Спочатку додавання, потім множення: (10 + 5) * 2 = 30
print(result)  # 30
